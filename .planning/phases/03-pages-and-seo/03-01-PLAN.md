  1. Install @astrojs/sitemap:
     ```bash

     ```

  2. Edit astro.config.mjs to add two things:
     - `site: 'https://parkermutsuz.com'` property in defineConfig (no trailing slash). This is a placeholder — the user should replace it with their actual production URL before deploying.
     - Import `sitemap` from `@astrojs/sitemap` and add `sitemap()` to the integrations array alongside the existing `mdx()`.

  The final config should look like:
  ```javascript
  import { defineConfig } from 'astro/config';
  import mdx from '@astrojs/mdx';
  import sitemap from '@astrojs/sitemap';
  import tailwindcss from '@tailwindcss/vite';

  export default defineConfig({
    site: 'https://parkermutsuz.com',
    integrations: [mdx(), sitemap()],
    vite: {
      plugins: [tailwindcss()],
    },
  });
  ```

  Do NOT use `npx astro add sitemap` — manual edit is cleaner per research findings.
  Do NOT add a trailing slash to the site URL — Astro concatenates site + route and a trailing slash produces double-slash URLs.
