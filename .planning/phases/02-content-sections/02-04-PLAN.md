<task type="auto">
  <name>Task 2: Photos gallery page with expandable sets and optimized images</name>
  <files>src/pages/photos/index.astro</files>
  <action>
Replace `src/pages/photos/index.astro` stub with the full photo gallery page:

1. Imports:
   ```
   import { Image } from 'astro:assets';
   import { getCollection } from 'astro:content';
   import BaseLayout from '../../layouts/BaseLayout.astro';
   ```

2. Data fetching and image resolution in frontmatter:
   ```typescript
   const photosets = await getCollection('photosets');

   // Eagerly import all photo assets — Vite resolves at build time
   const imageModules = import.meta.glob<{ default: ImageMetadata }>(
     '/src/assets/photos/**/*.{jpg,jpeg,png,webp}',
     { eager: true }
   );
   ```

3. Template structure:
   - `<h1>` heading: "Photos" with `text-5xl md:text-7xl font-bold tracking-tight`
   - Subtitle: "Photography and visual work." in `text-lg text-neutral-600 dark:text-neutral-400`
   - Below, render each photoset as a `<details>` element:
     ```
     <div class="mt-12 space-y-4">
       {photosets.map(set => (
         <details class="border-2 border-neutral-900 dark:border-neutral-100 group">
           <summary class="p-6 cursor-pointer font-bold text-xl select-none">
             {set.data.title}
             <span class="ml-2 text-sm font-normal text-neutral-500">
               {set.data.images.length} photos
             </span>
           </summary>
           <div class="p-6 pt-0">
             <p class="text-neutral-600 dark:text-neutral-400 mb-6">{set.data.description}</p>
             <div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
               {set.data.images.map(img => {
                 const mod = imageModules[img.src];
                 return mod ? (
                   <Image
                     src={mod.default}
                     alt={img.alt}
                     class="w-full h-auto"
                     widths={[400, 800]}
                     sizes="(max-width: 640px) 100vw, (max-width: 1024px) 50vw, 33vw"
                   />
                 ) : (
                   <p class="text-red-500 text-sm">Image not found: {img.src}</p>
                 );
               })}
             </div>
           </div>
         </details>
       ))}
     </div>
     ```
   - Handle empty state: if no photosets, show "No photo sets yet."
   - The error fallback (`Image not found: {img.src}`) is critical for debugging glob path mismatches — do NOT silently return null.

4. Pass title="Photos" and description="Photography and visual work." to BaseLayout

5. Key implementation notes:
   - `widths` and `sizes` props on `<Image>` enable responsive srcset generation
   - `eager: true` in the glob means all images are resolved at build time (no async needed)
   - The image `class="w-full h-auto"` ensures images fill their grid cell proportionally
   - No JS needed — `<details>/<summary>` handles expand/collapse natively
